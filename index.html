MES LANGUAGES MAITRISÉS

HTML

Le HTML, signifiant HyperText Markup Language, est le langage de base d’une page web. Avec son balisage bien précis, il représente le squelette d’un site. Il définit son contenu et sa structure. Le point central du HTML, également celui du web, sont les liens.

En savoir plus sur le HTML

CSS

Le CSS ou Cascading Style Sheet, est un des languages principaux du web. Il est utilisé conjointement avec le html pour définir le style d’une page web. Il peut créer des espacements pour rendre une page plus aérée ou des animations pour créer un peu d’action.

En savoir plus sur le CSS

JAVASCRIPT

Le Javascript est l’un des trois languages principaux du web et est la pierre angulaire d’un site interactif. Grâce à ses sturucture de contrôle, on peut réaliser toute sorte de chose pour rendre nos pages web animée.

En savoir plus sur le Javascript

PHP

Hypertext Preprocessor, plus connu sous l’abréviation PHP, est un langage utilisé côté serveur. Il permet via une serveur HTTP de rendre n’importe quel site dynamique. Ce langage est très utilisé dans le domaine à cause de sa rapidité et de sa flexibilité.

En savoir plus sur le PHP

SQL

Le SQL ou Stuctured Query Language est un langage permettant d’interagir avec une base de donnée relationnel. Il peut aussi bien récupérer des données que en modifié ou même, couplé avec d’autre langage, vérifier des informations.

En savoir plus sur le SQL

DART ET FLUTTER

Dart est un langage optimisé pour réaliser des applications disponible sur plusieurs plateforme. Durand mes étude, j’ai pu l’apprendre en utilisant Flutter. Il s’agit d’un kit de développement, open-source visant à nous aider à réaliser toute sortes d’application.

En savoir plus sur Dart

MES QUALITÉS

Calme

Le calme est nécessaire dans ce métier. Savoir garder son sang froid en toute circonstance est une qualité que j’ai développée au fils des années qui s’avère être utile.

Autonomie

L’autonomie est très importante. N’étant pas friand de contact sociaux étant plus jeune, même avec les professeurs, j’ai appris à me débrouillé par mes propres moyens.

Ponctualité

Je met un point d’honneur à être toujours à l’heure que ce soit au travail ou dans ma vie de tout les jours.

Sympathique

Même si on peut avoir du mal à le remarquer au premier abord, une fois que l’on apprend à me connaitre on s’aperçoit que je peut apporter de la bonne humeur et de la gaieté dans un environnement.
